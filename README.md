# about
【ホームページ】

### For Developers
Theme:https://themes.gohugo.io/themes/hugo-hero-theme/
```
$git clone -b gh-pages  https://github.com/BeginnersSec/about.git
$cd about
$hugo server
```
localhost:1313でローカルサーバーが起動

/content/ -> MD<br>
/themes/hugo-hero-theme/layouts/index.html -> index.html<br>

必要項目を編集後、
```
$hugo
```
で再ビルド
